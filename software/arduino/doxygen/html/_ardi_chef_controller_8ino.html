Detailed Description

ArdiChef's 'Main' Sketch used for all Arduino board(s).

Primary Functions:

  • Web Server - |Optionally| host EWS through ethernet shield
  • I2C Communications - Recieve, Parse, and Execute I2C Commands
  • Manage Ports - Load JSON, Start/Stop Motors, Cancel in Motion

Arduino Mega2560 Desloo Ethernet Shield w/5100 Wiznet (3) ULN2003 Step Motor Driver boards
